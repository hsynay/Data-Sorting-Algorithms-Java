# Data Sorting Algorithms in Java 🚀

Bu proje, temel sıralama algoritmalarının Java dilinde implementasyonlarını ve çalışma sürelerinin (nanosaniye cinsinden) karşılaştırmasını içerir.

## 🧠 Uygulanan Algoritmalar
* **Selection Sort:** O(n²) zaman karmaşıklığı, O(1) alan karmaşıklığı.
* **Insertion Sort:** Küçük veri setlerinde verimli, nested loop yapısı.
* **Bubble Sort:** En büyük elemanı sona taşıyarak ilerleyen swap mantığı.
* **Merge Sort:** Divide and Conquer (Böl ve Yönet) yaklaşımıyla çalışan verimli algoritma.
* **Recursive Insertion Sort:** Yinelemeli yaklaşım ile dökümante edilmiştir.

## 📊 Karşılaştırma
`Main.java` dosyası üzerinden kullanıcıdan seçim yapması istenir ve seçilen algoritmanın çalışma süresi milisaniye cinsinden ekrana yazdırılır.
